Project name : Project Zero Hour

team : Void Seekers

The project covers the full Incident Response Lifecycle, including:

    Preparation: Establishing incident handling procedures, defining roles, and setting up monitoring tools.
    
    Identification: Detecting suspicious activities through SIEM log analysis, threat intelligence, and security alerts.
    
    Containment: Isolating affected systems to prevent lateral movement and further compromise.
    
    Eradication: Removing malicious artifacts, closing exploited vulnerabilities, and ensuring systems are clean.
    
    Recovery: Restoring affected services and validating their operational integrity.
    
    Lessons Learned: Documenting the incident, analyzing root causes, and implementing security improvements.
